# HealthCare Data Website

Hi, I am Harsha.

I created this repository because I am working in healthcare, so I know how useful structured patient-style data can be when starting projects, building demos, testing dashboards, and exploring healthcare workflows.

In healthcare, even patient personal information and other important details are part of how many real systems are designed and connected. Because of that, I created this repository as a simple and helpful starting point for healthcare-related projects. It is made to be easy to use, easy to understand, and easy to extend.

This project includes:

- A healthcare website with synthetic patient records
- Patient detail pages
- HL7-style raw data files
- Structured data that can help you quickly start a project

You can use this repository directly for learning, demos, experiments, portfolio work, and project starters.

If you want to increase it, improve it, or build something bigger on top of it, please do it.

Thank you,
Harsha
